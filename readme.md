# Twitch Selenium Test

This is an E2E test project with Selenium, created by Karoly Puskas

## Demo

![Demo GIF](twitch_demo.gif)

## Structure

helpers/
Utility modules and helper functions for the test suite.

pages/
Page Object Model (POM) files representing different web pages.

screenshots/
Stores screenshots taken during test execution.

tests/
Contains test scripts and related cache.

conftest.py
Pytest configuration file for fixtures and hooks.

readme.md
Main project documentation.

twitch_demo.gif
Demo animation of the tests.
